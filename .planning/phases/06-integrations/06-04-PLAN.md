<objective>
Load the UpPromote tracking pixel on every page and wire referral code capture from URL
params through to checkout. Commission structure (Standard 10% / Silver 12% / Gold 15%,
Ambassador, Customer Referral 5%) and fraud rules (self-referral block, IP clustering,
14-day hold) are configured in the UpPromote dashboard — no theme code required.

Purpose: Without the tracking pixel and referral code pass-through, UpPromote cannot
attribute sales to affiliate links and discount codes will not apply at checkout.

Output: Pixel loads on every page; `?ref=CODE` captured to sessionStorage; checkout URL
includes the discount code so Shopify applies it automatically.
</objective>

<tasks>

- [ ] **Task 1: Inject UpPromote tracking pixel in theme.liquid**

  Files: `layout/theme.liquid`

  The UpPromote tracking pixel is a `<script>` tag that must load on every page, including
  the order confirmation page (for conversion attribution). Add in `<head>` after the GA4
  block:

  ```liquid
  {%- if settings.upromote_merchant_id != blank -%}
    <script async
      src="https://cdn.upromote.me/js/affiliate.min.js"
      data-merchant-id="{{ settings.upromote_merchant_id }}">
    </script>
  {%- endif -%}
  ```

  The `upromote_merchant_id` setting was already declared in `config/settings_schema.json`
  in plan 06-01. No additional schema changes needed.

  Also add the module script load before `</body>`:
  ```liquid
  {%- if settings.upromote_merchant_id != blank -%}
    <script type="module" src="{{ 'upromote.js' | asset_url }}"></script>
  {%- endif -%}
  ```

  Verify: `grep -c "upromote_merchant_id" layout/theme.liquid` returns >= 1.

- [ ] **Task 2: Create assets/upromote.js — referral capture + checkout wiring**

  Files: `assets/upromote.js`, `assets/cart-drawer.js`

  `assets/upromote.js`:

  On DOMContentLoaded:
  1. Read `ref` param from `new URLSearchParams(location.search).get('ref')`.
  2. If present and non-empty, store in `sessionStorage` under key `upromote_ref`.
  3. Also store in `sessionStorage` as `discount_code` (same value — UpPromote uses the
     affiliate's referral code as the Shopify discount code).

  Checkout URL wiring:
  - Query `document.querySelectorAll('a[href*="/checkout"]')` on DOMContentLoaded.
  - For each link, if `sessionStorage.getItem('discount_code')` is set, append
     `?discount=CODE` to the href (or `&discount=CODE` if query string already present).
  - Also dispatch a custom event `upromote:ref-captured` with `{ detail: { code } }` so
    cart-drawer.js can react if needed.

  In `assets/cart-drawer.js`:
  - Listen for `upromote:ref-captured` event on `document`.
  - When the checkout button's href is set (it is built dynamically from `/cart.js`
    response), append `?discount=${event.detail.code}` to the checkout URL.
  - This covers the case where cart-drawer renders after upromote.js fires.

  Commission tiers and fraud rules are fully dashboard-side — document in `06-CONTEXT.md`
  that the merchant must configure:
  - 3-tier commission: Standard 10%, Silver 12%, Gold 15%
  - Ambassador program
  - Customer Referral 5%
  - Self-referral block: enabled in UpPromote Settings → Fraud Detection
  - IP clustering threshold: set to 3 conversions per IP per 24h
  - Conversion hold: 14 days

  Verify: `assets/upromote.js` exists; `grep -c "upromote_ref" assets/upromote.js` >= 1;
  `grep -c "upromote:ref-captured" assets/cart-drawer.js` >= 1.
