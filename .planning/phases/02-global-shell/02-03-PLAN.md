    - Open: removes `-translate-x-full`, adds `translate-x-0` on `#mobile-nav-drawer`; removes `hidden` on `#mobile-nav-overlay`; sets `document.body.style.overflow = 'hidden'`; sets `aria-expanded="true"` on hamburger; focus trap: Tab cycles through `#mobile-nav-close` + all `<a>` inside drawer

    - Open: hides `#nav-links` (`opacity-0 pointer-events-none`), shows `#nav-search-bar` (`opacity-100`), calls `.focus()` on the input element
    - Close: reverses visibility; clears input value; returns focus to `#nav-search-toggle`
    - Both modules use `export default` and call their init function immediately (same pattern as custom-cursor.js)
  </behavior>
