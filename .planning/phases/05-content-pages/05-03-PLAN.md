  <action>
Create `assets/size-guide.js` — the fit recommender Vanilla JS module.

The module must:

1. Define a hardcoded size lookup table as a `const SIZE_TABLE` array of objects, ordered XXS → 3XL. Each entry: `{ size, bustMin, bustMax, waistMin, waistMax, hipMin, hipMax }` (all in inches, inclusive ranges). Values matching the static HTML table: XXS {28,31,22,24,30,33}, XS {32,34,25,27,34,36}, S {34,36,27,29,36,38}, M {36,38,29,31,38,40}, L {38,41,31,34,40,43}, XL {41,44,34,37,43,46}, 2XL {44,48,37,41,46,50}, 3XL {48,52,41,45,50,54}.

2. Define `findSize(bust, waist, hip)` function: iterate `SIZE_TABLE`. A row matches when bust falls within [bustMin, bustMax] (inclusive). If bust matches, check waist and hip too. If all three match → return `row.size`. If only bust matches → treat as bust-priority match (save as candidate). After full iteration, return first candidate (bust-priority) or `null` if no match.

   Input validation (D-02 + security requirement): before calling `findSize`, parse inputs with `parseFloat`. Validate each is a finite positive number greater than 0. If any input fails validation, do not proceed — show no-match message.

3. On `DOMContentLoaded`:
   - Get refs: `bustInput = document.getElementById('sg-bust')`, `waistInput = document.getElementById('sg-waist')`, `hipInput = document.getElementById('sg-hips')`, `submitBtn = document.getElementById('sg-submit')`, `resultDiv = document.getElementById('sg-result')`, `sizePill = document.getElementById('sg-size-pill')`, `noMatch = document.getElementById('sg-no-match')`.
   - Guard: if `!submitBtn` return early (section not present on page).
   - `submitBtn.addEventListener('click', () => { ... })`: parse and validate inputs (positive finite numbers, per security requirement); call `findSize`; if result → show `resultDiv` (remove `hidden`), set `sizePill.textContent = result`, hide `noMatch`; if null → hide `resultDiv`, show `noMatch` (remove `hidden`).
   - `[bustInput, waistInput, hipInput].forEach(input => input?.addEventListener('input', () => { resultDiv.classList.add('hidden'); noMatch.classList.add('hidden'); }))` — reset on any input change (per D-02 reset rule).

No external dependencies. No `fetch`. No DOM manipulation beyond the elements above.
