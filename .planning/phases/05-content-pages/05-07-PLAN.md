  <action>
Create `sections/page-content.liquid` implementing D-07.

This section is intentionally minimal — policy pages have no hero banner. Content comes entirely from the Shopify page object (`page.title`, `page.content`).

Structure:

```liquid
<section class="bg-cream">
  <div class="max-w-3xl mx-auto px-4 py-16">
    <h1 class="font-display text-4xl font-normal text-deep mb-8">{{ page.title | escape }}</h1>
    <div class="font-body text-sm text-mid leading-relaxed prose-headings:font-display prose-headings:font-normal prose-headings:text-deep">
      {{ page.content }}
    </div>
  </div>
</section>
```

Note: `{{ page.content }}` is NOT escaped. Shopify renders `page.content` as pre-sanitized rich text HTML — the same trust boundary as `product.description`. Adding `| escape` would double-encode the HTML and break formatting. This is intentional and matches D-07 spec.

`{{ page.title | escape }}` IS escaped — it is a plain text string, not sanitized HTML.

Schema: minimal — no settings needed (all content comes from `page.*` object). Schema name: "Page Content". Presets: one preset named "Page Content".

```json
{% schema %}
{
  "name": "Page Content",
  "settings": [],
  "presets": [
    { "name": "Page Content" }
  ]
}
{% endschema %}
```
