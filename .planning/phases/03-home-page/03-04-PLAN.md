## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-10 | Injection | post.caption.value output | mitigate | Use Liquid's default escaping; do not use raw filter on metaobject text fields |
| T-03-11 | Injection | post.product.value.product.url in href | mitigate | Internal Shopify product URL; Liquid auto-escapes URLs in href attributes |
| T-03-12 | Spoofing | social_post.product variant reference | accept | Only admin can create/edit metaobjects; not user-submitted data |
| T-03-13 | Tampering | npm/pip/cargo installs | accept | No package installs; pure Liquid files |
