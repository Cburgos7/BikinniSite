  <action>
    Create assets/testimonial-carousel.js as a Vanilla JS ES module following the mobile-nav.js pattern (named exports, no default export, init() as entry point).

    The module must:

    1. Export a named init() function.
    2. Inside init(): read the JSON data from document.getElementById('testimonial-data').textContent. Parse with JSON.parse. If parsing fails or data is empty or has fewer than 2 items, exit early (no carousel needed for 0 or 1 quotes).
    3. Grab DOM refs: sectionEl = document.getElementById('testimonial-section'), quoteEl = document.getElementById('testimonial-quote'), authorEl = document.getElementById('testimonial-author'), dotsContainer = document.getElementById('testimonial-dots').
    4. If any required element is null, return early.
    5. Build dot buttons: for each item in data, create a button element. Append to dotsContainer. First button gets active classes (w-4 h-2 bg-coral rounded-full transition-all duration-300), others get inactive classes (w-2 h-2 bg-mid rounded-full transition-all duration-300). Each button has aria-label "Go to testimonial N+1".
    6. State: let currentIndex = 0.
    7. showQuote(index) function:
       - Fade out: quoteEl.classList.add('opacity-0'), authorEl.classList.add('opacity-0').
       - After 500ms setTimeout: update quoteEl.textContent with data[index].quote. Update authorEl inner content: author text + optional " · location" if location is truthy. Fade back in by removing opacity-0.
       - Update dots: iterate all dot buttons, apply inactive classes to all, apply active classes to the one at index.
       - Update currentIndex = index.
    8. next() function: showQuote((currentIndex + 1) % data.length).
    9. Dot click handlers: each dot button calls showQuote(i) where i is its index.
    10. Auto-rotate: let timer = null. startTimer() = setInterval(next, 5000). stopTimer() = clearInterval(timer).
    11. IntersectionObserver on sectionEl: when intersecting (isIntersecting = true) call startTimer(); when leaving call stopTimer(). Use threshold: 0.
    12. Hover: sectionEl.addEventListener('mouseenter', stopTimer); sectionEl.addEventListener('mouseleave', startTimer).
    13. Touch: sectionEl.addEventListener('touchstart', stopTimer, { passive: true }); sectionEl.addEventListener('touchend', startTimer, { passive: true }).

    Keep the module self-contained. Do not import any external libraries.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| theme editor blocks → Liquid JSON output | Quote, author, location text written by store admin, emitted as JSON into page |
| JSON script tag → JS JSON.parse | JS reads admin-authored data already embedded server-side |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-14 | Injection | block.settings.quote piped through json filter | mitigate | Use Liquid's `json` filter for all block settings emitted into the JSON script tag — this escapes special characters |
| T-03-15 | Injection | JS reads JSON and sets textContent | mitigate | Use element.textContent (not innerHTML) when setting quote and author text in JS — no HTML injection possible |
| T-03-16 | Denial of Service | setInterval without IntersectionObserver guard | mitigate | Timer only runs while section is in viewport (IntersectionObserver) + stopTimer on mouseleave/touchstart |
  | T-03-17 | Tampering | npm/pip/cargo installs | accept | No package installs; pure Liquid + Vanilla JS |
</threat_model>
